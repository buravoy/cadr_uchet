# Кадровый учёт — Сотрудники предприятия

Практическая работа: модульное программирование в базах данных (вариант 3).

## Структура проекта

| Файл | Назначение |
|------|-----------|
| `main.py` | Точка входа, консольное меню |
| `database.py` | Работа с SQLite (CRUD) |
| `analytics.py` | Средний оклад по отделам |
| `seed.py` | Генератор тестовых данных |

## Запуск

```bash
python main.py
```
