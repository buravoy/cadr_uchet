# Кадровый учёт — Сотрудники предприятия

Практическая работа: модульное программирование в базах данных (вариант 3).

## Структура проекта

| Файл | Назначение |
|------|-----------|
| `main.py` | Точка входа, консольное меню |
| `database.py` | Работа с SQLite (CRUD) |
| `analytics.py` | Средний оклад по отделам |
| `seed.py` | Генератор тестовых данных |

## Запуск

```bash
python main.py
```

## Меню

```
1. Показать всех сотрудников
2. Редактировать сотрудника (или создать нового)
3. Удалить сотрудника
4. Средний оклад по отделам
5. Сгенерировать 10 тестовых записей
6. Удалить все тестовые данные
0. Выход
```
